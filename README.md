# SoundBridge

> 🎧 声音的桥梁 - 跨端音频融合软件

## 核心目标

**游戏时不用摘耳机，同时听电脑和手机的声音**

## ✨ 功能特性

### 🔗 连接方式
- ✅ WiFi 局域网（自动发现）
- ✅ WiFi 直连（热点模式）
- ✅ USB 有线连接
- ✅ 蓝牙连接

### 🎵 音频模式
- ✅ 均衡模式（50-100ms 延迟）
- ✅ 高音质模式（48kHz/24bit）
- ✅ 超低延迟模式（<30ms）
- ✅ 动态切换

### 🔄 传输方式
- ✅ 双向同时传输（手机↔电脑）
- ✅ 混音模式（两边音频混合播放）

### 🎛️ 控制方式
- ✅ Windows 桌面客户端
- ✅ Android App
- ✅ 系统托盘常驻
- ✅ 全局快捷键
- ✅ 双向控制

### ✨ 附加功能
- ✅ 设备自动记忆
- ✅ 启动自启
- ✅ 连接状态通知
- ✅ 回声消除（AEC）
- ✅ 噪声抑制（NS）
- ✅ 自动增益控制（AGC）
- ✅ 电平指示器

## 🛠️ 技术栈

| 组件 | Windows | Android |
|------|---------|---------|
| **语言** | C++ (核心) + C# (UI) | Kotlin + C++ (JNI) |
| **UI框架** | WinUI 3 | Jetpack Compose |
| **音频采集** | WASAPI | AAudio |
| **编解码** | libopus | libopus |
| **网络** | 自定义 UDP/QUIC | 自定义 UDP/QUIC |
| **回声消除** | WebRTC APM | WebRTC APM |
| **构建** | CMake + MSBuild | Gradle + NDK |

### 快捷键
| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Alt+M` | 切换混音模式 |
| `Ctrl+Alt+T` | 切换传输方向 |
| `Ctrl+Alt+P` | 暂停/恢复 |
| `Ctrl+Alt+S` | 打开设置 |

## 📊 性能指标

| 指标 | 目标 |
|------|------|
| **延迟（均衡模式）** | <100 ms |
| **延迟（超低延迟）** | <30 ms |
| **CPU占用（空闲）** | <2% |
| **CPU占用（传输中）** | <15% |
| **内存占用** | <100 MB |
| **带宽（每通道）** | 64-128 kbps |
